# ComfyUI-Logic-Switch

一个支持动态输入数量的随机字符串选择节点。支持1-20个字符串输入，每个都有独立的开关控制。也支持通过逗号分隔的字符串输入模式。

A random string selector node with dynamic input count. Supports 1-20 string inputs, each with individual enable switch. Also supports comma-separated string input mode.

Clone to `ComfyUI/custom_nodes/` and restart ComfyUI to use.
